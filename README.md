# 🛒 Optimisation de parcours en supermarché – PyQt & Algorithmique des Graphes

## 📌 Présentation du projet  

Ce projet, réalisé dans le cadre du **BUT Informatique**, permet d’optimiser les trajets dans un supermarché en utilisant **l’algorithmique des graphes** et une **interface graphique développée avec PyQt**.  

L'application permet de :  
- **Positionner des produits** sur un plan de magasin  
- **Générer un itinéraire optimal** pour récupérer une liste de courses  
- **Gérer l’organisation du magasin** à travers une interface intuitive  

Le projet est structuré en **architecture MVC** et a été développé en **groupe**, avec une gestion de projet via **Git**.  

---

## 🎯 Objectifs et fonctionnalités  

### 🔹 Application 1 – Gestion du magasin  
- Charger et afficher un **plan du magasin**  
- Définir une **grille de navigation ajustable**  
- Associer chaque produit à une **position précise**  
- Sauvegarder et charger les configurations  

### 🔹 Application 2 – Optimisation du parcours  
- Générer une **liste de courses aléatoire**  
- Afficher les **positions des produits** sur le plan  
- Calculer et afficher **le chemin optimal** pour collecter tous les articles  
- Prendre en compte les **contraintes d’accessibilité** des allées
